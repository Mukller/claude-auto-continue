# Claude Code Auto-Continue

Автоматически нажимает **Try again** / **Continue** в окнах Claude Code по расписанию — когда заканчиваются лимиты.

## Возможности

- **Расписание** — нажать кнопку в точное время (когда лимиты сбросятся)
- **Несколько чатов** — выбрать первые N окон Claude Code
- **Try again + Continue** — ищет обе кнопки, умеет нажать Enter после
- **Режим наблюдения** — повторять каждые N секунд
- **BFS по дереву доступности** — надёжно находит кнопки в Electron/Chrome приложениях
- Кольцевой таймер обратного отсчёта, тёмная тема

## Установка

```bash
pip install pyautogui pygetwindow pillow uiautomation
```

## Запуск

```bash
python claude_continue_gui.py
```

Или двойным кликом по `run.bat`.

## Использование

1. Выставить время сброса лимитов (например `05:00`)
2. Нажать **СТАРТ**
3. Программа ждёт, затем автоматически кликает **Try again** в выбранных окнах
4. После клика опционально нажимает **Enter** (отправить продолжение)

## Системные требования

- Windows 10/11
- Python 3.9+
