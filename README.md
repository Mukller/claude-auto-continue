# Claude Code Auto-Continue

Автоматически находит и нажимает кнопки **Try again** / **Continue** в Claude Code по расписанию — когда заканчиваются лимиты и сервер просит повторить запрос.

Работает через **поиск по скриншоту** (template matching) — наводит и кликает мышью по найденной кнопке на экране, как это сделал бы человек. Не зависит от внутреннего устройства Electron-окна Claude Code, поэтому работает надёжно независимо от версии приложения.

## Возможности

- **Расписание** — нажать кнопку в точное время (например, когда лимиты сбрасываются)
- **Поиск по скриншоту** — захватываешь область с кнопкой один раз, дальше программа сама находит её на экране
- **Несколько окон/чатов одновременно** — кликает по первым N найденным кнопкам (сортировка сверху-вниз)
- **Try again + Continue** — ищет обе кнопки параллельно, можно включать/выключать раздельно
- **Enter после клика** — после нажатия кнопки автоматически отправляет сообщение клавишей Enter
- **Режим наблюдения** — повторяет проверку каждые N секунд
- **Настройка точности** — ползунок confidence для template matching (на случай разного рендеринга/DPI)
- Кольцевой таймер обратного отсчёта, тёмная GitHub-тема

## Установка

```bash
pip install -r requirements.txt
```

(`pyautogui`, `pillow`, `opencv-python-headless` — последний нужен для точного сравнения с допуском, без него работает только точное пиксельное совпадение)

## Запуск

```bash
python claude_continue_gui.py
```

Или через `run.bat` (показывает консоль с ошибками, если что-то пошло не так).

Ярлык на рабочем столе использует `pythonw.exe` — открывается без консольного окна.

## Как пользоваться

1. **Захвати шаблоны кнопок** — нажми «📷 Захватить» рядом с «Try again» и/или «Continue». Экран затемнится, выдели рамкой кнопку на реальном диалоге Claude Code (можно открыть тестовый чат с лимитом или просто аккуратно обвести кнопку на скриншоте интерфейса).
2. Нажми **«🔍 Проверить сейчас»** чтобы убедиться, что шаблон находится на экране.
3. Выставь время и нажми **СТАРТ**.
4. Программа ждёт нужного времени, затем ищет кнопки на экране и кликает по первым N найденным (мышью, по-настоящему).
5. Если включено «Enter после клика» — после нажатия отправляет сообщение.

## Системные требования

- Windows 10/11
- Python 3.9+
