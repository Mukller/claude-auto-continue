# ⚡ Claude Code Auto-Continue

Автоматически находит и нажимает кнопку **Try again** в Claude Desktop по расписанию — когда сервер временно ограничивает запросы (rate limit). Умеет переключаться между несколькими чатами в сайдбаре одного окна и нажимать **Enter**, чтобы продолжить сессию, которая просто ждёт ввода после исчерпания лимита.

Кнопка ищется через **UI Automation по тексту** (Windows) или по **скриншоту-шаблону** (macOS). Electron-приложения часто не проставляют элементам правильную ARIA-роль, поэтому поиск идёт по видимому тексту на любом типе элемента.

## Возможности

- **Расписание** — сработать в точное время (например, когда лимиты сбрасываются ночью)
- **Plan запусков (циклы)** — задай несколько времён (05:00, 08:00, 17:00…), каждое отработает полный цикл. Опциональный ежедневный повтор или разовые запуски с автоудалением.
- **Автопоиск кнопки** — без ручного захвата шаблона; ищет «Try again» по тексту через UI Automation (Windows) или по скриншоту (macOS)
- **Переключение между чатами в сайдбаре** — заходит в выбранные чаты и обрабатывает каждый (Windows; на macOS — только текущий вид)
- **Два независимых действия на чат:**
  - `Try again` — находит и кликает реальную кнопку (ошибка временного ограничения сервера)
  - `Continue` — нажимает **Enter** после захода в чат, независимо от того, нашлась ли кнопка
- **Режим наблюдения** — повторяет проверку каждые N секунд
- **Резервный вариант** — если авто-поиск не находит кнопку, можно захватить шаблон кнопки по скриншоту
- **Тёмная и светлая тема**, переключение в один клик
- **Системный трей** и **уведомления рабочего стола** (опциональные зависимости)
- **Автозапуск** при входе в систему (Windows Registry / macOS LaunchAgent)
- Кольцевой таймер обратного отсчёта

## Установка

```bash
pip install -r requirements.txt
```

**Windows:** `pyautogui`, `pillow`, `uiautomation` — обязательны.  
**macOS:** `pyautogui`, `pillow` — обязательны; `uiautomation` — только Windows, не нужен.  
`opencv-python-headless` — только для резервного варианта (поиск по шаблону с допуском).

## Запуск

```bash
python claude_continue_gui.py
```

Или через `run.bat` (Windows, показывает консоль с ошибками) / `python claude_continue_gui.py` (macOS).

## Как пользоваться

1. Открой Claude Desktop, нажми **«↻ Найти»** в приложении — покажет найденное окно и список чатов.
2. Отметь нужные чаты галочками в списке.
3. Отметь нужные действия — `Try again` и/или `Continue`.
4. Нажми **«🔍 Проверить сейчас»**, чтобы протестировать поиск кнопки.
5. Выставь время и нажми **СТАРТ**.

> **macOS:** переключение по чатам недоступно (нет UI Automation). Приложение активирует Claude через `osascript` и нажимает кнопку в текущем видимом чате.

| Поиск кнопки Try again | ✅ UI Automation + шаблон | ✅ шаблон |
| Автозапуск | ✅ Registry | ✅ LaunchAgent |

## Известные ограничения

- Если окно Claude перекрыто другим окном и `bring_to_foreground` не срабатывает — клик может попасть не в то место. Программа логирует предупреждение.
- Резервный поиск по шаблону чувствителен к масштабу/теме — обновляй шаблон при смене темы или DPI.
- `Continue` жмёт Enter «вслепую» — если в поле ввода уже что-то напечатано, это будет отправлено.

## Системные требования

- Windows 10/11 или macOS 12+
- Python 3.9+
